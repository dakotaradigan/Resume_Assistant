# Ben AI - Benchmark Eligibility Chatbot

Personal project (2024) demonstrating full-stack AI capabilities.

## What It Does
AI chatbot helping financial advisors quickly determine benchmark eligibility for client portfolios.

## Tech Stack
FastAPI, OpenAI GPT-4, Pinecone (vector search), WebSockets, JavaScript

## Key Features
- Semantic search across financial benchmark documentation
- Iterative function calling with safety guardrails
- Real-time WebSocket communication
- Modular frontend with Claude-inspired UI

## Impact
Reduced advisor research time from 30+ minutes to under 2 minutes per query.

## Technical Highlights
- Built RAG architecture with Pinecone vector database
- Implemented connection management and WebSocket fallback to REST
- Designed intent-based function routing for consistent query handling
- Created production-ready error handling and logging

This project showcases ability to build end-to-end AI applications with modern web technologies.
